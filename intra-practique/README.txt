Créer une page HTML (bouffe.html) que va afficher le titre « Gestion des bouffe ». Vous y avez 10 images d’bouffe avec leur noms. Dans un fichier JavaScript bouffe.js, vous allez créer un tableau tabBouffe ayant les informations suivantes pour chaque bouffe : nom, origine (indienne, americaine, italienne), catégorie (boisson, cote, repas).

Votre application doit répondre aux questions suivantes: 

1. Afficher un « select » avec le nom de tous les bouffes. Lorsque vous sélectionnerez un bouffe, vous allez afficher dans l’ordre le nom de l’bouffe, suivit de sa photo et près celle-ci sa origine et sa catégorie.

2. Afficher un « select » avec l’origine de tous les bouffes. Lorsque vous sélectionnerez une origine, vous allez afficher seulement le nom de tous les bouffes de cette origine.
Ex: indienne -> biryani, cari, lassi, samosa
