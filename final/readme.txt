Créer une page HTML  
    -index.html 
qui va afficher le titre « Gestion des livres » suivi après de votre nom et prénom. 

Créer 3 boutons dont le texte de chacun sera « Question 1 », « Question 2 » et « Question 3 ». Lorsque vous allez cliquer sur un de ces boutons le code
correspondant sera exécuté. Ce code se dans une fonction nommée function questionX() {…}, où X correspond au numéro de la question annoncée plus bas. 
Toutes ces fonctions seront dans le fichier livres.js.

Les questions: 

1. Afficher tous les données du tableau tabLivres, avec un titre « Liste des livres » ainsi que le nom des
colonnes et après les données.

2. Afficher un champ « input type= » dans lequel vous allez lire le « auteur » (code de l’auteur). Vous allez
afficher dans le même format que la question 1. 

3. Afficher un champ « input type= » dans lequel vous allez lire la « catégorie ». Vous allez afficher
seulement les titres de tous les livres de cette catégorie dont « année » (année de publication) est comprise 
entre 1980 et 1992 inclusivement.
